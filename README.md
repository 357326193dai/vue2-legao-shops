## NOTE

- 乐高商铺端后台管理平台，分桌面端与手机端
- 基于 vue2+webpack4，实现全程无刷新操作，体验第一
- 布局采用 vw+rem+flexbox
- 使用技术 vue2、es6/7、vuex、vue-router、element-ui、axios、scss、svg 等等...
- 实现功能挺多，一个完整的后台系统，包含路由、单数据流、权限，换肤，语言、等等...
